# フォーム自動入力スクリプト

## セットアップ

```bash
pip install -r requirements.txt
```

## 使い方

### 1. data.json を編集

```bash
# または自分のエディタで開く
notepad C:\Users\taked\Desktop\form-automation\data.json
```

### 2. スクリプト実行

```bash
python C:\Users\taked\Desktop\form-automation\script.py
```

---

## data.json フォーマット

### 必須フィールド

| フィールド | 説明 | 例 |
|-----------|------|-----|
| `url` | フォームのURL | `https://sinmido.com/contact/` |
| `name` | 名前 | `山田太郎` |
| `furigana` | フリガナ | `ヤマダタロウ` |
| `company` | 会社名 | `株式会社テスト` |
| `phone` | 電話番号 | `09012345678` |
| `email` | メールアドレス | `test@example.com` |
| `inquiry_content` | お問い合わせ内容 | `テスト送信です。` |
| `inquiry_items` | お問い合わせ項目（複数選択可） | `["HP制作・改善", "新卒採用全般"]` |

### 任意フィールド

| フィールド | 説明 | 例 |
|-----------|------|-----|
| `department` | 部署 | `営業部` |

---

## inquiry_items の選択肢

### 採用・教育
- `求める人が採用できる仕組みにしたい`
- `採用の為の充分な応募を確保したい`
- `応募してきた方の惹きつけを強化したい`
- `内定後や承諾後のフォローを充実させたい`
- `採用活動で発生する工数を削減したい`
- `社員の育成、教育を強化したい`
- `人事制度の構築や組織改善をしたい`

### マーケティング・集客
- `HPからの資料請求やイベント予約を増やしたい`
- `Instagram広告やGoogle広告で成果の出る手法を知りたい`
- `新規集客のための広告宣伝計画の立て方を知りたい`
- `検索結果で上位表示させたい･AI検索で表示されるようにしたい`
- `SNSの運用やイベント企画の業務を軽減したい`
- `集客ターゲットの来場を増やしていきたい`
- `お客様に見せる資料を用意したい･刷新したい`
- `自社HPも他社のようにスタイリッシュにしたい`

### その他
- `その他`

## 複数項目選択例

```json
{
  "inquiry_items": [
    "HP制作・改善",
    "新卒採用全般",
    "WEB広告"
  ]
}
```

---

## 注意事項

- `inquiry_content` に改行を入れる場合は `\n` を使用してください
- 電話番号とメールアドレスは必ず有効な形式を入力してください
- スクリプト実行時はブラウザが自動で開きます
